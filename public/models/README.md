# Muscular anatomy model

`muscular.glb` is a web-optimized derivative of the muscular anatomy model
published by Adamas Advanced Designs. The source model is derived from
BodyParts3D and informed by Z-Anatomy.

- Source and attribution: https://www.adamasdesigns.com/licenses
- BodyParts3D: https://dbarchive.biosciencedbc.jp/en/bodyparts3d/
- Z-Anatomy: https://www.z-anatomy.com/
- Asset license: CC BY-SA 4.0

The model was simplified and compressed with glTF Transform for Recomp. The
asset remains licensed under CC BY-SA 4.0; Recomp's source code remains MIT.
